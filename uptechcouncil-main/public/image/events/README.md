# Event Images

Place event images for the homepage Events and Activity section here.

**Expected filenames:**
1. `pakistan-business-summit-davos-2025.png`
2. `uk-pakistan-business-summit-2025.png`
3. `dha-peshawar-uk-road-show-2025.png`

**Recommended specifications:**
- Format: PNG (or JPG)
- Resolution: 800x450 or higher (16:9 aspect ratio recommended)
- Style: Professional event photography or branded event graphics
- File size: Optimize to under 300KB per image for performance

**Usage:**
Images are displayed in event cards on the homepage with a gradient overlay for text readability and "Past Event" badges.
